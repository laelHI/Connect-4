# Features:

Simple Connect 4 that has:
- Adjustable board sizes 
- Game win values that change based on board sizes
- Randomized computer mode to play against a computer
- Two-player mode to play with someone next to you
- Play again feature

# Technicalities 
- Three different java files
- One for common methods
  - This was for methods that could be used in main file but didnt really really rely on game logic. Used to simplfy the main file
- One for game logic
  - This is filled with methods relate to or rely on the game and logic of game. An example could be the check wins method of print board method
- One to run the game
  - Final and main file where the code is run and the main loop of the game takes place
